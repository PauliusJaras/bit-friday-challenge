## 🏃 Run locally
Would like to run this project locally? Open terminal and follow these steps:

1. Clone the repo
https://github.com/PauliusJaras/bit-friday-challenge

2. Install NPM packages
`npm i -D dead-server`

3. Run the server
`npm run dev`
